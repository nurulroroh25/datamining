Weather Event Data Documentation

State

The state name where the event occurred (no State ID’s are included here; State Name is spelled out in ALL CAPS). See details 

Example: GEORGIA

State FIPS

A unique number (State Federal Information Processing Standard) assigned to the county by the National Institute for Standards and Technology (NIST). 

Example: 45

CZ FIPS

The county FIPS number is a unique number assigned to the county by the National Institute for Standards and Technology (NIST) or NWS Forecast Zone Number. See details

Example: 245, 003, 155

Source

The source reporting the weather event (can be any entry; isn’t restricted in what’s allowed).

Example: Public, Newspaper, Law Enforcement, etc.

Magnitude

The measured extent of the magnitude type ~ only used for wind speeds (in knots) and hail size (in inches to the hundredth).

Example: 0.75, 60, 2.75

Magnitude Type

The type of magnitude measurement (wind gust, sustained wind, etc.).

Example: EG, MS, MG, ES

Range

Distance (to the tenth of a mile) to the geographical center or primary post office of a particular village/city, providing that the reference point is documented in the Storm Data software location database table.

Example: 0.59, 4.84

Azimuth

16-point compass direction from the reference point documented in the Storm Data software location database table of > 130,000 locations.

Example: ENE, WSW

Location

The name of city, town, or village from which the range is calculated and the azimuth is determined. 

Example: ASHEVILLE, SAN DIMAS

Latitude

The latitude where the event occurred (Signed negative (-) if it’s in the southern hemisphere).

Example: 31.25, 32.76

Longitude

The longitude where the event occurred (Signed negative (-) if it’s in the western hemisphere).

Example: -93.97, -94.52
